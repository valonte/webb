Mapping for Land Developers for the City of Toronto


Ideal

A mobile-friendly interactive web map that compares and visualizes different neighborhoods according to education, crime rates, income, and living costs which can help developers, city planners, non-government officials identify potential areas of concern.


Reality

There is no mobile-friendly and interactive web map that displays, aggregate, and compare these factors together.


Consequences

It is hard for users to visualize areas using tabular data directly available from Toronto’s Open Data Portals. Therefore, allocation.


Proposal

Create an interactive web map that displays the 4 factors above and where aggregation or comparison tools for the neighborhoods are present.
